 Sales Analytics Dashboard
Project Overview
This project is an interactive sales analytics dashboard built using Python and Streamlit. It analyzes retail sales data and provides insights through visualizations and key metrics.

Features
 Total Sales, Profit, and Orders (KPI metrics)
 Sales by Category and Region
 Top 10 Products Analysis
 Interactive dashboard using Streamlit

Tools & Technologies
 Python
 Pandas
 Matplotlib
 Seaborn
 Streamlit

How to Run
1. Install dependencies:
   pip install streamlit pandas matplotlib seaborn

2. Run the dashboard:
   streamlit run dashboard.py
